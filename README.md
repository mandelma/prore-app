# ProRe App

ProRe App is a modern service marketplace platform designed to connect service providers and customers. The goal of the project is to support small businesses by making it easier for professionals to offer services and for customers to find trusted providers.

The application is being developed as a full-stack solution with both web and mobile support.

## Features

### User Management
- User registration and authentication
- Secure login and account management
- Role-based functionality for service providers and customers

### Service Marketplace
- Create and manage service listings
- Search and browse available services
- Connect service providers with customers

### Real-Time Communication
- Integrated chat system
- File sharing between users
- Image sharing between users
- Direct communication between service providers and customers

### Maps and Location Services
- Google Maps integration
- Location-based service discovery
- Custom map interactions and functionality

### Media Management
- Image upload and storage using AWS Cloud Services
- Secure access to uploaded images

### Calendar Integration
- Calendar functionality for scheduling and availability management

### Internationalization
- Multi-language support
- Language switching functionality (currently implemented on the landing page and under active development)

### Responsive Design
- Fully responsive user interface
- Optimized for desktop, tablet, and mobile devices

- 
### Mobile Application
- Mobile app development in progress
- Built using Capacitor for cross-platform deployment

## Installation

Clone the repository:

```bash
git clone https://github.com/mandelma/prore-app.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
